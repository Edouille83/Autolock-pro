# AutoLock Pro — Reparetonphone83

Application de référence serrurier automobile.

**👉 [Ouvrir l'application](https://edouille83.github.io/autolock-pro/)**

## Fonctionnalités
- 200+ véhicules avec transpondeur, lame, litchi, super chip Xhorse
- Workflow Marque → Modèle → Année → Infos complètes
- Lien direct MK3 pour chaque véhicule
- Calculateur de prix intervention
- Historique des interventions
- Mode jour/nuit
- Fonctionne hors-ligne (PWA)

## Installation sur iPhone
1. Ouvrir le lien dans Safari
2. Bouton Partager → "Sur l'écran d'accueil"
3. L'app s'installe comme une vraie application
